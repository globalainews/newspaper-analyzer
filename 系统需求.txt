## 1. 视频生成页签需求细节

### 1.1 页面布局与结构
- **左右分栏布局**：左右各占50%宽度
- **左侧区域**：新闻列表和操作控制
- **右侧区域**：图片预览和矩形框调整
- **最小尺寸**：左侧容器最小宽度450px

### 1.2 新闻列表功能
- **显示格式**：每个新闻条目一个文本框
- **字体设置**：12号字体，提高可读性
- **操作按钮**：
  - 删除按钮：支持删除单个新闻
  - 上下移动按钮：调整新闻顺序
- **交互功能**：
  - 点击新闻条目时，在右侧图片中高亮显示对应的矩形框
  - 支持选中状态显示

### 1.3 数据管理功能
- **自动加载**：进入视频生成页签时，如果存在JSON文件，自动加载新闻列表
- **数据来源**：从分析结果文件（.txt）生成JSON数据
- **保存机制**：点击"生成数据"时覆盖现有JSON文件
- **文件路径**：`analysis_results/{图片名}_video.json`

### 1.4 图片预览与编辑功能
- **矩形框调整**：
  - 8个调整手柄（4个角 + 4条边中点）
  - 支持拖拽移动整个矩形框
  - 支持调整矩形框大小
  - 实时预览调整效果
- **交互特性**：
  - 鼠标滚轮缩放图片
  - 支持拖拽移动图片位置
  - 点击新闻条目时显示对应的矩形框

### 1.5 控制按钮功能
- **生成数据**：从分析结果生成视频数据JSON文件
- **保存数据**：保存编辑后的新闻内容和位置信息
- **生成视频**：启动视频生成流程

### 1.6 进度显示功能
- **进度条**：显示视频生成进度
- **状态标签**：显示当前处理步骤
- **后台处理**：使用线程避免界面卡顿

## 2. 视频输出要求细节

### 2.1 视频规格
- **视频格式**：竖屏视频
- **分辨率**：1080x1920（9:16比例）
- **帧率**：24fps
- **编码**：libx264，无音频

### 2.2 视频内容结构

#### 2.2.1 顶部标题区域
- **报纸名称**：根据文件名自动识别（华尔街日报/金融时报）
- **字体设置**：微软雅黑，60号字体，白色
- **位置**：顶部居中，距离顶部50px

#### 2.2.2 日期显示区域
- **内容格式**：年月日 + 星期（中文显示）
- **字体设置**：微软雅黑，36号字体，金色（#FFD700）
- **位置**：标题下方，距离顶部130px

#### 2.2.3 报纸图片区域
- **布局要求**：宽度适应屏幕，放在上半部分
- **高度计算**：总高度1920px - 标题200px - 字幕400px = 1320px
- **位置**：居中显示，顶部留出200px空间给标题

#### 2.2.4 字幕区域
- **位置要求**：报纸图片下方，确保在视频范围内
- **最大位置**：底部150px内
- **自适应调整**：如果字幕位置超出范围，自动调整报纸图片高度

### 2.3 字幕显示要求
- **内容来源**：新闻的中文内容（chinese_content）
- **字体设置**：微软雅黑，40号字体，白色
- **显示方式**：逐个显示，每条新闻显示3秒
- **动画效果**：
  - 淡入效果：0.5秒
  - 淡出效果：0.5秒
  - 实际显示时间：2秒（总时长3秒）
- **字幕框大小**：视频宽度减去100px，高度120px

### 2.4 背景效果
- **渐变背景**：两层颜色叠加
  - 底层：深蓝色（0, 0, 30）
  - 上层：紫色（30, 0, 60），50%透明度
- **持续时间**：与视频总时长一致

### 2.5 时间控制
- **每条新闻时长**：3秒
- **视频总时长**：`max(15秒, 新闻数量 × 3秒)`
- **字幕开始时间**：每条新闻开始后0.3秒显示

### 2.6 中文字体处理
- **字体路径**：
  - 首选：`C:\Windows\Fonts\msyh.ttc`（微软雅黑）
  - 备用：`C:\Windows\Fonts\simhei.ttf`（黑体）
- **字体检测**：自动检测字体文件是否存在

### 2.7 错误处理与进度显示
- **错误提示**：在终端打印详细错误信息，方便调试
- **进度回调**：实时更新进度条和状态文字
- **完成提示**：视频生成完成后弹出成功消息框

### 2.8 文件输出
- **默认文件名**：`news_video_{日期}.mp4`
- **保存路径**：用户选择保存位置
- **格式选项**：MP4文件格式

# 报纸头版下载与分析工具 - 需求规格文档

## 1. 项目概述
**项目名称**：报纸头版下载与分析工具  
**项目类型**：Python桌面应用程序  
**技术栈**：Python + Tkinter + Google Gemini API + MoviePy  
**主要功能**：报纸头版下载、AI分析、视频生成

## 2. 核心功能需求

### 2.1 报纸下载功能
- **下载源**：从 kiosko.net 下载报纸头版图片
- **支持报纸**：
  - 华尔街日报 (Wall Street Journal)
  - 金融时报 (Financial Times)
- **下载选项**：
  - 今日报纸
  - 昨日报纸  
  - 自定义日期
- **网络配置**：
  - 支持HTTP代理（默认端口1080）
  - 金融时报提供备用下载URL
- **文件管理**：
  - 图片保存到 `downloaded_images/` 子目录
  - 文件名格式：`{报纸代码}_{日期}.jpg`

### 2.2 图片分析功能
- **AI模型**：Google Gemini 2.5 Flash
- **分析内容**：
  - 识别报纸头版中的主要新闻块
  - 提取每个新闻块的矩形位置坐标
  - 提取新闻标题和内容（中英文）
  - 识别整个报纸内容区域
- **输出格式**：结构化JSON数据
- **结果保存**：分析结果保存到 `analysis_results/` 目录

### 2.3 用户界面需求
- **主界面**：标签页式布局
- **标签页结构**：
  1. **首页**（原"分析图片"）：图片列表、预览、分析结果
  2. **视频生成**：新闻编辑、视频预览、生成控制
- **界面布局**：
  - 左侧：功能控制区
  - 右侧：内容展示区
  - 支持全屏显示

### 2.4 图片预览功能
- **预览特性**：
  - 支持鼠标滚轮缩放
  - 支持拖拽移动
  - 显示新闻块矩形框
  - 支持矩形框调整（8个调整手柄）
- **交互功能**：
  - 点击新闻块高亮显示
  - 实时调整矩形框位置和大小

### 2.5 视频生成功能
- **视频规格**：
  - 竖屏格式：1080x1920
  - 支持中文字幕
  - 渐变背景效果
- **视频内容**：
  - 顶部：报纸名称 + 日期星期
  - 中部：报纸头版图片
  - 底部：新闻内容字幕（逐个显示）
- **生成控制**：
  - 进度条显示
  - 后台线程处理
  - 错误处理机制

### 2.6 数据管理功能
- **新闻编辑**：
  - 可修改新闻内容
  - 支持删除新闻条目
  - 支持调整新闻顺序
- **数据保存**：JSON格式保存编辑结果
- **自动加载**：进入视频生成页签时自动加载已有数据

## 3. 技术规格

### 3.1 配置文件
```json
{
  "gemini": {
    "api_key": "YOUR_GEMINI_API_KEY",
    "model": "gemini-2.5-flash",
    "proxy": {"host": "127.0.0.1", "port": 1080}
  },
  "download_settings": {
    "save_directory": "downloaded_images",
    "image_quality": "750"
  },
  "export_settings": {
    "export_directory": "E:\\中文听见\\报纸头版"
  }
}
```

### 3.2 分析提示词（prompt.txt）
- 要求识别报纸内容区域和新闻块
- 输出结构化JSON格式
- 包含中英文内容提取
- 指定坐标位置信息

### 3.3 模块化架构
- **main.py**：主程序入口
- **downloader.py**：下载功能模块
- **analyzer.py**：分析功能模块  
- **video_generator.py**：视频生成模块
- **config_manager.py**：配置管理模块
